# WebAF

WebAF is a modular Python framework for authorized web application security assessments.

## Features

- Asset discovery
- JavaScript endpoint analysis
- Endpoint inventory
- Plugin architecture
- HTML reporting
- JSON reporting

## Modules

- Headers
- Cookies
- CORS
- HTTP Methods
- Swagger/OpenAPI
- robots.txt
- security.txt
- XXE

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python3 webaf.py http://target
```

## Disclaimer

Only use against systems you own or are authorized to assess.
